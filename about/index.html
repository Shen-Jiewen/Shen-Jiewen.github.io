关于我

个人简介

你好！我是Javen，一名热爱技术和分享的开发者。这个博客是我记录学习、项目经验和技术探索的地方，同时希望能够帮助和启发其他开发者。

博客的使命

我的初衷是通过写作和分享：

  • 记录学习和项目中的技术问题与解决方案。
  • 分享代码实践、工具使用及开发心得。
  • 探索新兴技术趋势，并尝试用简洁的方式讲解它们。

希望我的博客能成为你学习道路上的伙伴，让我们一起进步！

我的技能栈

  • 编程语言：Python, C/C++, Java
  • 开发领域：嵌入式开发，机器人控制，数据分析
  • 常用工具：Keil，VSCode，Matlab/Simulink
  • 其他：对 Linux 系统、开源项目和机器人仿真感兴趣

博客的年份

  • [2022] : 开始了我的开发学习之路
  • [2024] : 正式搭建了这个个人博客

你能在这里找到什么

  • 开发教程：从嵌入式开发到控制算法的学习教程。
  • 项目展示：分享我在工作或个人时间中的开发项目。
  • 学习笔记：总结书籍、课程以及线上资源的学习心得。
  • 技术趋势：关注新技术的发展，并尝试分享我的见解。
